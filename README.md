# 🌐 Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **CSS**. This project showcases my skills, projects, and contact information in a clean and modern layout.

## 🚀 Features

- Responsive design
- Clean and modern UI
- About Me section
- Skills section
- Projects showcase
- Contact section
- Easy to customize

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
portfolio/
│── index.html
│── style.css
│── images/
│── assets/
└── README.md
```

## 💻 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/simrandhiman043/portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
